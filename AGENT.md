# AGENT.md

This repository hosts `mcp-use`, a full-stack framework for building with the Model Context Protocol (MCP), enabling the creation of MCP servers, clients, and AI agents in both Python and TypeScript.

**Key Technologies/Frameworks**: Model Context Protocol (MCP), Python, TypeScript, LangChain, OpenAI, Playwright, React.

**Main Features**: The framework facilitates the development of AI agents capable of multi-step reasoning and tool use, direct MCP client connections, and custom MCP server creation. It includes a web-based debugger (MCP Inspector), UI widgets for interactive applications, multi-server support, real-time streaming, and built-in observability and OAuth support.

**Architectural Patterns**: The project utilizes a monorepo structure with distinct Python and TypeScript implementations, adhering to a client-server and agent-based architecture with a modular design for its core components, CLI, inspector, and scaffolding tools.
